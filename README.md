# Sistema de Gestión de Calificaciones (Java)

Proyecto desarrollado en Java que permite gestionar alumnos y sus calificaciones mediante un menú interactivo.

## Funcionalidades
- Agregar alumnos con su calificación
- Mostrar alumnos registrados
- Calcular promedio general
- Mostrar alumnos aprobados
- Interfaz mediante JOptionPane

## Tecnologías
- Java SE
- Programación Orientada a Objetos
- ArrayList

## Estructura
- principal: control del flujo
- logica: reglas del sistema
- alumno: modelo de datos
